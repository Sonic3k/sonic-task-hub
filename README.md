## Getting Started

### Backend (Spring Boot)

```bash
cd sonic-task-hub-api
mvn spring-boot:run
```

The backend will start on http://localhost:8080

### Frontend (React)

```bash
cd sonic-task-hub-web
npm install
npm run dev
```

The frontend will start on http://localhost:3000
